# 类介绍：

**LogManager类：** 该类是一个静态工具类，用于对日志的输出
- info() 正常日志输出打印
- warn() 警告日志输出打印
- error() 错误日志输出打印

**Game类：** 该类包含整个游戏逻辑的函数，初始化游戏、更新、渲染、输入监听等游戏循环的主要的类
- Game() 构造函数，初始化，加载资源
- ~Game() 构析函数，释放内存，清理对象
- run() 启动游戏
- update() 游戏更新
- render() 游戏渲染
- playAudio() 播放游戏音频音效
- input() 用户输入
